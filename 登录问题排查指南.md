# 注册和登录问题排查指南

## 问题诊断

如果您无法注册或登录，请按照以下步骤操作：

## 解决方案

### 1. 清除浏览器缓存

**Chrome/Edge:**
- 按 `Ctrl + Shift + Delete`
- 选择"缓存的图片和文件"
- 点击"清除数据"

**Firefox:**
- 按 `Ctrl + Shift + Delete`
- 选择"缓存"
- 点击"清除"

### 2. 强制刷新页面

- 按 `Ctrl + F5` (Windows)
- 或 `Ctrl + Shift + R` (Windows)

### 3. 使用隐私/无痕模式

打开浏览器的隐私/无痕模式访问：
- Chrome: `Ctrl + Shift + N`
- Edge: `Ctrl + Shift + N`
- Firefox: `Ctrl + Shift + P`

然后访问: http://localhost:3000

### 4. 检查浏览器控制台

1. 按 `F12` 打开开发者工具
2. 切换到"Console"(控制台)标签
3. 尝试注册或登录
4. 查看是否有红色错误信息

### 5. 使用测试页面

访问测试页面（绕过可能的缓存问题）:
http://localhost:3000/test-frontend.html

这个页面可以直接测试注册和登录API。

### 6. 验证服务器状态

确保服务器正在运行：
- 应该能看到："服务器运行在 http://localhost:3000"

## 快速测试

在浏览器控制台中运行以下代码测试API：

```javascript
// 测试注册
fetch('/api/auth/register', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    username: 'testuser',
    email: 'test@example.com',
    password: 'Test@123'
  })
})
.then(r => r.json())
.then(d => console.log('注册结果:', d))
.catch(e => console.error('错误:', e));

// 测试登录
fetch('/api/auth/login', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    email: 'test@example.com',
    password: 'Test@123'
  })
})
.then(r => r.json())
.then(d => console.log('登录结果:', d))
.catch(e => console.error('错误:', e));
```

## 常见错误及解决

### 错误: "Failed to fetch"
- 原因: 服务器未运行
- 解决: 确保运行了 `node server-memory.js`

### 错误: "网络错误"
- 原因: 浏览器缓存或CORS问题
- 解决: 清除缓存或使用无痕模式

### 页面加载空白
- 原因: JavaScript文件缓存
- 解决: 强制刷新 (Ctrl+F5)

### 注册后无法跳转
- 原因: localStorage问题
- 解决: 清除浏览器数据

## 如果以上方法都无效

1. 重启服务器：
   - 停止当前服务器 (Ctrl+C)
   - 重新运行: `node server-memory.js`

2. 更换浏览器试试

3. 检查防火墙设置
