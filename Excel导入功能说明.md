# Excel批量导入功能说明

## 概述

Excel批量导入功能允许管理员快速将大量物品数据导入库存系统，支持.xlsx和.xls格式。

---

## 物品属性定义

每个物品包含以下属性：

| 属性 | 字段名 | 必填 | 说明 | 示例 |
|------|--------|------|------|------|
| **编号** | itemNo | 否 | 自定义编号 | 001, 002 |
| **名称** | itemName | **是** | 物品名称 | 青眼白龙 |
| **编码** | itemCode | 否 | 产品编码 | LOB-001 |
| **稀有度** | rarity | 否 | 稀有程度 | 极稀有、超稀有、稀有等 |
| **类型** | itemType | 否 | 物品类型 | 卡牌、补充包、盒装等 |
| **数量** | quantity | **是** | 拥有数量 | 3, 5, 10 |
| **价格** | value | 否 | 单价（元） | 150.00 |
| **状态** | condition | 否 | 物品状态 | 近完美、良好等 |
| **描述** | description | 否 | 备注信息 | 经典稀有卡 |

## 使用方法

### 方法一：通过Web界面导入

1. **登录系统**
   - 以管理员或超级管理员身份登录

2. **进入库存管理**
   - 点击"库存管理"按钮

3. **打开导入对话框**
   - 点击"Excel导入"按钮

4. **下载模板（可选）**
   - 点击"下载模板"获取标准格式
   - 模板包含示例数据和正确的列名

5. **准备Excel文件**
   - 按照模板格式填写数据
   - 确保"名称"和"数量"列有值
   - 其他列为可选

6. **上传并导入**
   - 选择Excel文件
   - 点击"开始导入"
   - 等待导入完成

7. **查看结果**
   - 成功：显示导入统计
   - 失败：显示错误详情和行号

### 方法二：通过API导入

```bash
curl -X POST http://localhost:3000/api/inventory/import \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -F "file=@test_inventory.xlsx"
```

```

itemNo | itemName | itemCode | rarity | itemType | quantity | value | condition | description
```

## 文件限制

- **支持格式**: .xlsx, .xls
- **文件大小**: 最大 5MB
- **单次导入**: 建议不超过1000行
- **编码**: UTF-8

## 错误处理

### 常见错误及解决

| 错误信息 | 原因 | 解决方法 |
|---------|------|---------|
| 缺少物品名称 | 名称列为空 | 填写名称列 |
| 数量不能为负数 | 数量为负值 | 修改为非负数 |
| 价格不能为负数 | 价格为负值 | 修改为非负数 |
| 无效的物品类型 | 类型值不在枚举中 | 使用有效的类型值 |
| 无效的稀有度 | 稀有度值不在枚举中 | 使用有效的稀有度值 |

### 部分成功处理

- 成功的行会被导入
- 失败的行会跳过
- 返回详细的错误报告
- 可以根据错误报告修正后重新导入

## 最佳实践

### 1. 数据准备
- 使用模板作为起点
- 在Excel中先验证数据
- 确保必填列有值
- 检查数值列格式

### 2. 分批导入
- 大量数据分批导入
- 每批100-500行为宜
- 确认成功后再导入下一批

### 3. 数据验证
- 导入前在Excel中检查
- 使用数据验证功能
- 避免特殊字符

### 4. 备份数据
- 导入前导出当前库存
- 保留原始Excel文件
- 记录导入批次

## 注意事项

1. **首次使用**
   - 建议先下载模板了解格式
   - 用小批量数据测试
   - 确认导入结果正确

2. **数据覆盖**
   - 导入不会覆盖现有数据
   - 每次导入都是新增物品
   - 如需更新请使用编辑功能

3. **性能考虑**
   - 大批量导入可能需要时间
   - 保持网络连接稳定
   - 不要在导入过程中关闭页面

4. **语言支持**
   - 列名支持中英文
   - 枚举值支持中英文
   - 系统会自动映射

## 故障排除

### 问题1：看不到"Excel导入"按钮
**原因**: 当前用户不是管理员
**解决**: 使用管理员账号登录

### 问题2：导入后数据为空
**原因**: Excel格式不正确或没有必填数据
**解决**: 检查列名和必填字段

### 问题3：部分数据导入失败
**原因**: 某些行数据不符合要求
**解决**: 查看错误报告，修正后重新导入

### 问题4：文件上传失败
**原因**: 文件格式不支持或超过大小限制
**解决**: 使用.xlsx或.xls格式，确保小于5MB
