# 默认超级管理员账号

## 🎯 快速使用

### 步骤 1：启动服务器
```bash
npm run demo
```

### 步骤 2：注册超级管理员
打开浏览器访问 http://localhost:3000

**重要**：由于是内存版本，**第一个注册的用户**会自动成为超级管理员！

请使用以下凭据注册（或任意你想用的用户名）：

```
用户名: admin
邮箱: admin@tcg.com
密码: admin123456
```

### 步骤 3：登录并使用
注册成功后，系统会自动登录并显示管理面板。

## 💡 提示

1. **内存版本特性**
   - 重启服务器后所有数据会清空
   - 每次重启后，第一个注册用户会成为超级管理员

2. **MongoDB 版本**
   - 数据持久化保存
   - 使用 `npm run create-admin` 创建超级管理员
   - 需要安装 MongoDB

3. **验证超级管理员权限**
   - 登录后应该看到"管理面板"标题
   - 有"用户列表"标签页
   - 有"创建用户"标签页
   - 有"角色管理"标签页（仅 superadmin）

---

## 🔐 安全提醒

生产环境中请：
1. 修改默认密码
2. 使用 MongoDB 持久化存储
3. 设置强 JWT_SECRET
4. 启用 HTTPS
