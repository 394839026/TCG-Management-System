## 🎯 核心功能

### 用户功能
1. **查看库存** - 浏览自己的所有物品
2. **添加物品** - 录入新的收藏品
3. **编辑物品** - 更新数量、价格等信息
4. **删除物品** - 移除不再需要的物品
5. **查看统计** - 了解收藏总价值和分布

### 管理员功能
1. **查看所有用户库存** - 了解系统内所有物品
2. **修改任意物品** - 帮助纠正错误信息
3. **管理用户收藏** - 维护系统数据质量

## 💡 使用方式

### 启动系统
```bash
npm run quick
```

### 访问库存页面
1. 打开浏览器 http://localhost:3000
2. 注册/登录账号
3. 点击"我的库存"或"库存管理"按钮

### 添加第一个物品
1. 点击"添加物品"按钮
2. 填写信息：
   - 物品名称：青眼白龙
   - 类型：卡牌
   - 数量：3
   - 状态：近完美
   - 单价：150
3. 点击"保存"
